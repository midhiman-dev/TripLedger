# TripLedger
TripLedger is an offline-first Progressive Web App (PWA) purpose-built for the Indian road-trip traveller.  It solves a single, deeply felt pain point: the inability to track shared trip expenses in areas with  unreliable or absent mobile connectivity — a condition that defines the majority of scenic travel  corridors across India (Hampi, Spiti, Coorg, Ladakh, and beyond).
